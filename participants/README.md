# 👥 Participants

This folder holds one file per workshop participant.

## How to add yours

1. Create a new file named `<your-name>.md` (lowercase, no spaces — use hyphens if needed)
2. Use the template below
3. Commit it on your feature branch
4. It gets merged in as part of Exercise 5

## Template

```markdown
# Hi, I'm <Your Name>! 👋

**GitHub:** @<your-github-username>
**Fun fact:** <one sentence about yourself>
**Why I joined the Coding Club:** <one sentence>

## What I want to learn
- <goal 1>
- <goal 2>
```
